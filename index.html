Align With Margin

Align w' Margin Code

Sass Code

Description

This Align Method uses the margin top and left properties. A fixed width and height of an element and the element's parent. The parent, in order to have a "nice" and easier align, must be 3 times bigger, in height and width, than the element.

Align The Element

In order to align the element in the x axis. We only have to set the margin-left property to these values.

  • Left Align: Just set it to 0.
  • Center Align: Set it the width of the element or a 3rd part of the parent's width.
  • Right Align: Set it to the double of the width of the element or 2 * 3rds of the parent's width.

Its almost the same process and math to align the element in the y axis, only changing the width of the element and its parent for height.

History

The Margin property was included in the first version of CSS in 1996, but I couldn't find any record if it was before the CSS standard, in different style sheets like the ViolaWWW style sheet.

Align With Table Cell

Align w' Tabe Cell Code

Sass Code

Description

This Align Method uses the table cell display for the parent element and the inline, block or not, display. For the alignment of the element, we area going to use the text-align and vertical-align properties for x and y axis alignment.

Align The Element

To align the element on any axis, we only have to change the respective property to the value or align that we want. The values of each align are pretty simple in this align method, its only the name of the position on the property.

The property of the x axis is text-align, and for the y axis is vertical-align.

    X Axis

  • left
  • center
  • right

    Y Axis

  • top
  • middle
  • bottom

History

The Table Objects are one of the most old thing in the web, being the base of the style and design of the old web. The table objects were in the web before the CSS standard, but with first version of CSS we could align the content of the tables.

Align With Positions

Align w' Positions Code

Sass Code

Description

This Align Method uses the position property. And the left, top and transform (translate) properties to align the element in the x and y axis.

Align the Element

For the x axis align, we are going to use the left property. For the y axis we are going to use the top property. And depending the align, we are going to use these values.

  • left / top: 0%
  • center: 50%
  • right / bottom: 100%

If we leave it like this, the element will be off the position by the same percentage. Example, with 50% it'll be off 50% from the center. To solve that, we are going to use the transform property with the translate function value.

The translate function needs two parameters, the translation on the x axis and the translation on the y axis. We are going to set the needed parameters just like the left and top properties, but we are going to set them as a negative percentage, adding the minus symbol before the percentage.

History

The position, left, top, right and bottom properties were introduced in the second version of CSS in 1998. But the transform property and translate value function were introduced in 2008, it was a new age for the web, at the dawn of new technologies such as Flexbox.

Align With Flexbox

Align w' Flexbox Code

Sass Code

Description

This Align Method uses Flexbox and its align properties, justify-content and align-items. Some old browsers such as old versions of Internet Explorer, might not support Flexbox.

Align The Element

Flebox is simple, almost like the table cell align method, it only uses the name of the align position as a value. But the names change a bit. For the x axis align, we are going to use justify-content and align-items for the y axis.

  • Left / Top: flex-start
  • Center: center
  • Right / Bottom: flex-end

History

Flexbox was introduced in 2009, but it wasn't the Flexbox of nowdays, it was a different version of flexbox a very weird and bad one. But in 2011, Flexbox changed to the version which everybody knows and uses today. Flexbox was also introduced as a part or "module" of the new third version of CSS.

Align With Grid Layout

Align w' Grid Code

Sass Code

Description

This Align Method uses one of the newest innovations in web development. CSS Grid Layout. Some old browsers such as old versions of Internet Explorer, might not support CSS Grid Layout.

We could use Grid to align the element just like Flexbox, but that isn't the main usage of Grid Layout. We are going to use the grid-template, row and column properties.

Align The Element

We are going to create a 3x3 grid template in the parent element, creating 3 cells in the row and the column. Then we can align the element on the x and y axis, setting the grid-row and grid-column properties to the number of cells that we need.

History

The idea of a Grid in CSS is old as tables in HTML. They were an idea, but not a real layout in CSS; sometimes the layout of grid was imitated using properties like floats and elements like the tables. The Floats are not in this list because of the inability of vertical and horizontal alignment. Sometimes using Javascript to recreate the Grid Layout. But it wasn't until 2017 that we could use one of the modules from the third version of CSS. CSS Grid Layout.
